# <div align="center"><a href="https://boafeira.com.br">Boafeira</a></div>

E-commerce simples para venda de produtos alimentícios

## Motivação

Não havia um e-commerce para produtos alimentícios disponível que fosse bom, rápido e entregasse onde moro, portanto decidi fazer o meu próprio e estou disponibilizando para que qualquer um consiga utilizar na sua localidade ou migrar para entregas nacionais algum dia.

A aplicação tem foco em privacidade, simplicidade e eliminar as maiores dores de quem vai fazer a feira em supermercados físicos ou precisa pedir por um aplicativo e esperar um entregador desconhecido ir entregar suas compras.
